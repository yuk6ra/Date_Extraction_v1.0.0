# Date_Extraction_v1.0.0

This is "Date Extraction".You can determine if the data you enter is a date.

# example
input 
The data is in date.txt. If you want to add data, please add a new line.
```
2020/01/05
2020/1/5
2020年1月5日
2020-1-5
2020/1/5
2020.1.5
2020/20/20
2020 1 5
2020 01 05
1995w44w47
Thank you
1998/33/52
3020/1/1
```

output

```
2020 1 5
2020 1 5
2020 1 5
2020 1 5
2020 1 5
2020 1 5
2020 1 5
2020 1 5
```
